# vue开发
>使用vue-cli2.9.x来搭建基础项目的架构，便于快速进行项目开发。

[vue代码风格指南](https://cn.vuejs.org/v2/style-guide/)
## pc搭建项目

### 技术栈
+ vue2.5.x
+ vue-router
+ vuex
+ axios

### 完成功能
+ 基础架构开发
+ 弹窗封装
+ 简单验证登录和退出登录

[下载项目地址](https://github.com/ymblog/vue-template)
